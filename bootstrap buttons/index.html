Custom Bootstrap Buttons

Customization: This button has a custom green background color (#4CAF50) and border, with a darker shade of green on hover. This is achieved by overriding Bootstrap's default button styles.

Customization: The corners of this button are rounded by applying a high border-radius: 50px; to create a pill-shaped button.

Customization: This button has extra padding (padding: 10px 40px;) and a larger font size (font-size: 1.2rem;) for emphasis.

Customization: This button changes color to tomato (#FF6347) when hovered over and to crimson (#DC143C) when clicked or active.

Bootstrap Feature: The btn-lg class enlarges this button compared to the default size, making it suitable for prominent actions.

Bootstrap and FontAwesome: This button includes a checkmark icon using FontAwesome to enhance the visual appeal and clarify its action.
